Welcome to Crew Manager BETA

Thank you for volunteering to test Crew Manager. This is unfinished software and you may experience bugs, errors, problems, data loss, etc.

The great news is that you can help squash those bugs:

  • The easiest way to report an issue is to click the "Feedback" button at the bottom right of each page - this will allow you to include a screenshot and a description of what went wrong
  • You can also raise an issue on Github

If you have feature requests rather than bug reports please raise these

  • On the forum ...
  • ... or on Discord

Crew Manager is open source so if you have the ability then I would also encourage you to dive into the codebase on Github and make your own amendments via a Pull Request.

An "Equipment Type" is the type aircraft that you fly - each different aircraft type (eg a Boeing 747) counts as a different type of equipment, but each airline may have unique seat layouts and capacities, all of which can be modelled in Crew Manager.

Crew Manager offers a number of built in equipment types, and also offers you offers you unlimited flexibility to create bespoke equipment types.

A "Flight Plan" is a plan for a particular flight.

It will have a departure and destination airport, a particular equipment type and a passenger manifest.

Each time you fly a particular route you will create a new flight plan for that particular flight.

Once you have created a flight plan you can fly the route.

In order to do this you need to download the Find My Plane client and run it on the local machine which is running Microsoft Flight Simulator.

Once you download the client you will be provided with a 6-digit IDENT code such as ABCDEF which you can then use to start the flight by choosing the Flight Plan and pressing connect.

Latest updates
14 May 2021 - New server

We had a couple of hours of planned downtime today as we migrated to a new server. Things are running much more smoothly now on a larger server which should handle the traffic better.

All your data should be intact, but if you have any issues please raise a bug on the Discord.

10 May 2021 - Deboarding passengers

You may have noticed that while passengers happily board your plane, they often struggle to get off again.

Deboarding of passengers is now supported by asking your crew to "start deboarding" or "deboard passengers".

9 May 2021 - Interactive tour

Squashing some bugs today, but also adding new functionality.

The most noticeable new addition in terms of features will be the option to have an interactive tour of both the inflight dashboard and the inflight crew communication page. I realised that these were pretty difficult to work out, so hopefully this will help new users get up to speed.

8 May 2021 - Beta open

Welcome aboard! I have opened a very limited beta program in the hope of getting some feedback and finding some bugs.

Please do report any bugs you find using the red box on the bottom of each page.
